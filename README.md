## 原版机制修改细节

### 世界生成

本章详细介绍关于世界生成的修改

#### 地形

 - 海平面大幅拉低到y=0
 - 取消地下洞穴、峡谷、地牢、矿道、试炼密室
 - 溶洞、繁茂洞穴和深暗之域生成调整
 - 地表平滑生成，且生物群系面积扩大两倍，包括河流宽度也扩大
 - 矿石仅以团簇的方式生成在接近地表的地方

#### 生物

 - 移除僵尸、骷髅、史莱姆、蝙蝠、末影人、苦力怕
 - 女巫仅在沼泽小屋生成or仅伴随掠夺者生成？
 - 蜘蛛仅在深色橡木林生成？

### 熔炉矿物熔炼

 - 熔炉分为火炉和炉膛两个方块，火炉产生高热量（仅接受煤炭之类），炉膛需要接受热量，均无gui
 - 烧炼矿物后产生金属流体，需要从炉膛导出冷却为发热的金属锭（导出会一次性清空炉膛，多余的流体会直接销毁
 - 发热的金属锭再次冷却为金属锭或在铁砧加工为装备头
 - 熔炉烧炼支持合金配方，放入不同矿物可以按比例烧炼出合金。
 - 部分高级配方需要更高温度，由此需要风箱加热

### 烟熏炉熏制

 - 烟熏炉类似熔炉分为两部分，一个发烟的炉和上面的支架，炉仅支持木料，支架需要下方的炉发烟
 - 根据使用的木料不同可以给食物熏制不同的风味！

### 烹饪

 - 炖锅和炒锅放在产热方块上产生不同食物（均无gui），完成后需要用碗盛出
 - 制造灶台可以右键营火或者地狱岩或者岩浆块形成，营火灶台可以接受木头燃料，地狱岩直接打火石点燃即可，岩浆块可以接受熔岩流体作为燃料
 - 烧烤架可以架在营火上

### 酿造

 - 发酵桶消耗被发酵物品来产生发酵液体，或发酵后物品，不同发酵流程可能需要多次发酵或者途中需要坩埚处理
 - 发酵桶可以制作发酵饮料，带来简单但有副作用的药水效果
 - 坩埚处理分为坩埚底下放置产热方块有无，需要手动搅拌，完成药水炼制，若上方有蒸馏釜则直接输出到蒸馏釜内
 - 混合釜，可以代替坩埚，消耗烈焰粉，完成药水炼制，若上方有蒸馏釜则直接输出到蒸馏釜内
 - 蒸馏釜，需要产热方块或者下方有被加热的（坩埚/混合釜/蒸馏釜），向上输出加强的药水

### 织布

 - 羊毛方块、地毯等需要使用织布机纺织（无gui）
 - 织布机可以放入最多三种不同颜色的羊毛（物品）来获得不同颜色组合的羊毛方块
 - 线需要通过织布机来获得

### 染色

 - 染料可以加入坩埚然后加热放入物品进行染色
 - 改变陶瓦颜色需要上釉然后烧制

### 附魔

 - 附魔台

### 铁砧与锻造台与装备制作

## 新增机制

### 豆腐

 - 添加石磨用于磨豆腐
 - 添加厨台用于加工

### 矿脉

 - 矿石以云状生成，其中生成1-2个矿脉核心，其余为普通矿石，镐子敲碎后获得1-2个粗矿
 - 矿脉核心无法直接破坏，也无法被活塞移动，需要在临近方块上放置产热方块（火炉等只能向上产热，使用专用的产热方块可以向四周产热）吸热，多个产热方块可以加速
 - 吸热完毕的矿脉核心遇水直接爆炸产生大量粗矿，或者由采矿方块采集（采矿方块每次可以采集大量粗矿，但都有概率使矿脉核心爆炸，越高级爆炸概率越低）
 - 矿石总生成在地表（-10~20）附近
 - 矿石（铜铁锡铝铅锌镍、冰晶石，煤炭，青金石，红石）

### 电力传输

 - 使用电线杆和无方块碰撞箱的电线连接，具有损耗，需要建造变压器来减少损耗，遵循电压与电流关系

### 蒸汽发电

 - 转子需要通入蒸汽旋转，产出水，转子周围需要再包围一圈磁铁才能发电
 - 蒸汽由锅炉放在产热方块上，向上产生蒸汽
 - 流体管道传输蒸汽或水

### 矿物增产

 - 洗矿：原始洗矿槽需要顶部有流动水，然后向下输出洗过的矿石，后有电动洗矿，可以获得精矿，精矿烧炼获得二倍产出
 - 磨粉：精矿粉碎后获得矿粉，可用于化学操作
